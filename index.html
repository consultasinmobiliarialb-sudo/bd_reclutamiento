📋 Instrucciones de Sincronización
Apps Script (sincronización bidireccional)
  1. Abrí tu Google Sheets → Extensiones → Apps Script
  2. Borrá el código existente y pegá el script descargado
  3. Ejecutá la función configurarHojas()
  4. Clic en Implementar → Nueva implementación
  5. Tipo: Aplicación web · Acceso: Cualquier usuario
  6. Copiá la URL /exec y pegala en la app
⚠️ Importante: Cada vez que modifiques el script, creá una nueva implementación (no actualices la existente) para que los cambios tomen efecto.
